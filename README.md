## Ludens site (MVP)

### Dev

- Start dev server: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`

### Content & i18n

See `CONTENT.md`.

### Pay page (Premium Tokens)

Static pay pages live in `public/pay/` (spec: `SPEC_PAY_PAGE.md`).

Local preview:

- with Next dev: open `http://localhost:3000/pay/index.html?acc=Test`
- or serve `public/pay/` with any static server

This is a Next.js (App Router) + Tailwind v4 project.
